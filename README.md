# HR-Recruitment-Bajaj-Finance
Developed Excel and Power BI dashboards for recruitment analytics, campus hiring, and MIS reporting. Tracked candidate progress across hiring stages, monitored hiring performance, and contributed to an offer decline reduction project through data-driven insights.

## 1. Campus Hiring Analytics Dashboard
#### Project Overview

Developed an Excel and Power BI dashboard to analyze campus hiring performance across multiple colleges and states. The dashboard tracks the complete hiring funnel from student registration to joining and helps the recruitment team identify high-performing colleges for future hiring drives.

#### Key Features
- College-wise and state-wise hiring analysis
- Registration, assessment, interview, offer, and joining tracking
- Conversion rate analysis across hiring stages
- Top-performing college identification
- Interactive Power BI visualizations and filters
##### Business Impact
- Enabled the campus hiring team to make data-driven decisions by identifying colleges with higher conversion rates, improving hiring efficiency, and optimizing campus recruitment strategies.

#### Tools Used
- Excel
- Power BI


## 2. Recruitment Funnel Dashboard
### Project Overview

Created a recruiter-wise recruitment dashboard to monitor candidate movement across 16 recruitment stages, from sourcing to joining. The dashboard provides real-time visibility into hiring activities and recruiter performance.

#### Key Features
- Recruiter-wise candidate tracking
- Stage-wise recruitment funnel analysis
- Real-time hiring pipeline monitoring
- Candidate status tracking across all recruitment stages
- Performance reporting and hiring insights
#### Business Impact

Improved recruitment visibility, enabled better workforce planning, and helped stakeholders monitor hiring progress and recruiter productivity effectively.

#### Tools Used
- Excel
- Power BI


## 3. Daily Open Positions vs Joined Tracker
### Project Overview

Designed a daily MIS reporting solution to track newly opened positions, offers released, and candidate joining status. The dashboard helps compare hiring demand against actual fulfillment.

#### Key Features
- Daily position opening tracking
- Offer release monitoring
- Joining status reporting
- Department-wise hiring progress analysis
- Trend analysis for hiring demand and fulfillment
#### Business Impact

Provided management with clear visibility into hiring progress, helping teams identify hiring gaps and take timely actions to meet workforce requirements.

#### Tools Used
- Excel
- Power BI

## 4. Offer Decline Reduction Analytics
### Project Overview

Contributed to a business-critical project aimed at reducing candidate offer decline rates. Generated upcoming joiner lists, analyzed candidate responses collected by the calling team, and identified candidates at risk of not joining.

### Key Features
- Upcoming joiner tracking
- Candidate response analysis
- Logic-based risk classification (Red Cases and Green Cases)
- Recruiter-wise risk reporting
- Joining conversion monitoring
### Business Impact

Helped recruiters proactively engage at-risk candidates, improve candidate experience, increase joining conversions, and reduce overall offer decline percentages.

### Tools Used
- Excel
- Power BI
